Some Tips and Tricks to help with efficient programming

  • To replace a list inplace use list[:],

    Example: nums = [1,2,3,4]; nums[:] = [x*2 for x in nums]; This list comprehension will replace the original nums and not create a copy.

  • Always use dictionary if possible. Much efficient than loops.

    For example: To check if count of characters is same in string and substrings. Use dict and check if string and substring dicts are same

  • Comparing integer as strings works the same as comparing integers!

    For example: 4 < 5 is the same as str(4) < str(5)

    Quick Review of Binary Search

    If you're trying to find the location of first occurence in array, use '<' (bisect left) in condition, else '<=' (bisect right)

    General Template

    Binary Search - Lower Bound (Bisect left): Smallest index such that arr[index] >= target

    Binary Search - Upper Bound (Bisect right): Smallest index such that arr[index] > target
